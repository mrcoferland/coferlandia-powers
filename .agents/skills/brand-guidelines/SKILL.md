# Brand Guidelines

## Note
Adapted for Coferlandia from generic visual-identity patterns; no third-party branding or trademarks are required.

## When to use this skill
Use this skill when you need to make an artifact look coherent across pages, slides, exports, or generated visuals.

## Principles
- Prefer one clear visual system over mixed styles.
- Keep contrast high and text readable.
- Use a limited palette with one accent color.
- Keep spacing, alignment, and hierarchy consistent.
- Reuse the same typography choices across related artifacts.
- Avoid imitating another company's exact brand look unless the user explicitly owns those assets and asks for it.

## Practical guidance
### Colors
- Choose a dark neutral, a light neutral, and one accent color.
- Use the accent color sparingly for emphasis, actions, or section markers.
- Make sure charts and diagrams still work in grayscale.

### Typography
- Pick one heading style and one body style.
- Use bold and size contrast before adding more fonts.
- Keep body text legible in print and on screen.

### Layout
- Use consistent margins and gutters.
- Align repeated elements the same way across pages.
- Leave enough whitespace for scanning.

## Output checklist
- consistent palette
- readable typography
- clear hierarchy
- no stray brand references
- no low-contrast combinations
