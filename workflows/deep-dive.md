# Deep Dive Workflow

Purpose: convert a candidate opportunity into a decision-quality report.

1. Restate the opportunity and decision needed.
2. Expand research with source links and dates.
3. Run `source-quality-scoring` on important evidence.
4. Run `market-analyst` for demand, competition, timing, and distribution.
5. Run `cfo-unit-economics` for ranges, scenarios, risks, and opportunity cost.
6. Run `builder-mvp` for the smallest validation experiment.
7. Run `skeptic-risk` for adversarial objections and decision pressure.
8. Resolve contradictions and document assumptions.
9. Run `opportunity-reporting` with the standard Opportunity Report format.
10. End with BUILD, TEST, WAIT, or KILL.

Safety: if the idea requires unsafe actions, mark KILL or require human review.
